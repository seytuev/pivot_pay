# Vertex - Исламское финансирование для бизнеса

Корпоративная платформа исламской рассрочки для крупных покупок под залог имущества. Часть экосистемы STV Group.

## Концепция

Vertex специализируется на финансировании крупных покупок для бизнеса по модели Мурабаха:
- Компания выбирает необходимое оборудование/актив
- Vertex выкупает актив у поставщика
- Продает клиенту с наценкой в рассрочку
- Обеспечение: залог имущества, доли компании, активов владельца
- Система поручителей и программа лояльности

## Целевая аудитория

- Производственные компании
- Торговые организации
- Сервисные предприятия
- Предприниматели, нуждающиеся в дорогостоящем оборудовании

## Что финансируем

- Коммерческий транспорт
- Складская техника
- Медицинское оборудование
- IT-инфраструктура
- Строительная техника

## Особенности

- B2B фокус (корпоративные клиенты)
- Крупные суммы (от 1 млн рублей)
- Залоговое обеспечение
- Соответствие принципам шариата (без риба)
- Система поручителей
- Программа лояльности
